El siguiente es un proyecto realizado en el lenguaje Java, en Eclipse. Posee las funcionalidades necesarias para realizar una compresión y descompresión de imagenes con tamaño 2000x2500 pixeles.

Las compresiones realizables son Huffman y Run Lenght Coding.

Para usar el programa, se debe importar a Eclipse el proyecto "TPE Teoria" y ejecutar "src/segundaParte/InterfazGrafica.java".
